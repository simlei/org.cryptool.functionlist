Welcome to CrypTool Functionlist’s documentation!¶

This document gives a high-level overview over the project. It introduces how to run it and how to interact with it. In the following, the reader is assumed to already know the CrypTool project.

Aim of this project is to generate the data that feed into the dynamic table of cryptological functions of the CrypTool project. To that end, it processes legacy files from, and data that is dynamically generated by:

  • CrypTool 1

  • CrypTool 2

  • CrypTool Online

  • JCrypTool

In a nutshell: The main program supports a –help parameter and is run as follows with default steps:

> ./src/flistapp.py

Requirements and Installation¶

  • Python 3.8 is required

  • pip is convenient to install the dependencies of the project as follows:

> pip install -r <path-to-this-project>/requirements.txt
